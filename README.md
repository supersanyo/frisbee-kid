## Getting Started
### Prerequisites
The script relies on two tools for the conversion.
1. [HandBrake](https://handbrake.fr/) - Rip DVD into MP4 videos
2. [VLC](https://www.videolan.org/) - Strip MP4 audio into MP3

### Windows
#### Get Handbrake & VLC
Folder setup
```
frisbee-kid
  \->handbrake
     \->HandBrakeCLI.exe
        libdvdcss-2.dll
    vlc
     \->vlc.exe
        (other vlc related files)
    fly-frisbee.ps1
    scan-frisbee.ps1
```
